 Project Overview
This project demonstrates deploying a static website on an AWS EC2 Ubuntu instance using the Nginx web server.

Technologies Used
- AWS EC2
- Ubuntu Linux
- Nginx
- Git
- GitHub
- SSH
  
 Features
-Created an EC2 instance
- Configured Security Groups
- Connected using SSH
- Installed and configured Nginx
- Hosted a static HTML website
- Version controlled using Git and GitHub

Commands Used
- sudo apt update
- sudo apt install nginx
- sudo systemctl status nginx
- df -h
- free -h
- ps aux

Output
The website is accessible using the EC2 Public IP.
